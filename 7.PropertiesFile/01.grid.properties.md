# What is grid.properties?

grid.properties is a configuration file that controls:

Whether tests run **locally** or on **Selenium Grid**

Which browser, platform, and version to use

Where the Grid Hub is located

Think of it as:

“Instructions for Selenium on where and how to run tests.”


### example:

```
remoteRun=false
urlGrid.Hub=http://localhost/wd/hub
targetPlatform=WINDOWS
versionBrowser=ANY
```

## 1️⃣ remoteRun=false

Decides where tests will run.

| Value   | Meaning                              |
| ------- | ------------------------------------ |
| `false` | Run tests **locally** (your machine) |
| `true`  | Run tests on **Selenium Grid**       |

Local run → fast debugging

Grid run → cross-browser & parallel testing

### framework view code is
```
if (remoteRun) {
    // use RemoteWebDriver
} else {
    // use local ChromeDriver / FirefoxDriver
}
```

## 2️⃣ urlGrid.Hub=http://localhost/wd/hub

The address of Selenium Grid Hub

When remoteRun=true, Selenium needs to know:

“Where is the Grid Hub?”

localhost → same machine

/wd/hub → Selenium’s default endpoint


**Hub**: Central controller

**Node**: Machine where browsers actually run


### example:

new RemoteWebDriver(new URL(urlGridHub), capabilities);

## 3️⃣ targetPlatform=WINDOWS

Specifies which operating system the browser should run on.

Common Values:

WINDOWS

LINUX

MAC

ANY

Selenium Grid may have nodes on:

Windows

Linux

macOS

This tells the Grid:

“I want my browser on Windows”

### example in code:

```
DesiredCapabilities caps = new DesiredCapabilities();
capabilities.setPlatform(Platform.WINDOWS);
```

## versionBrowser=ANY

Specifies browser version

| Value    | Meaning                  |
| -------- | ------------------------ |
| `ANY`    | Any available version    |
| `latest` | Latest installed version |
| `114`    | Specific version         |

Avoid failures caused by strict version matching

Useful when Grid nodes have different versions


### example in code:

```
capabilities.setBrowserVersion("ANY");
```


# real time

```

if (remoteRun) {
    DesiredCapabilities caps = new DesiredCapabilities();
    caps.setPlatform(Platform.valueOf(targetPlatform));
    caps.setVersion(versionBrowser);

    driver = new RemoteWebDriver(
        new URL(gridHubUrl), caps
    );
} else {
    driver = new ChromeDriver();
}
```
