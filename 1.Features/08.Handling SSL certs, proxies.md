## 1. Handling HTTPS certificates in Selenium with Chrome (Java)

### Accept insecure certificates

```
# here ChromeOptions is a class provided by selenium. in this we have methods to handle certs and we need to pass variable as an argument in webdriver class
ChromeOptions options = new ChromeOptions();
options.setAcceptInsecureCerts(true);

WebDriver driver = new ChromeDriver(options);
driver.get("https://expired.badssl.com/");
```

## 2. Handling HTTPS certificates in Selenium with Firefox

```
FirefoxOptions options = new FirefoxOptions();
options.setAcceptInsecureCerts(true);

WebDriver driver = new FirefoxDriver(options);
driver.get("https://your-https-site");
```

## 3. Handling HTTPS certificates with RemoteWebDriver (Selenium Grid / Docker)

```
ChromeOptions options = new ChromeOptions();
options.setAcceptInsecureCerts(true);

WebDriver driver = new RemoteWebDriver(new URL("http://grid:4444/wd/hub"), options);
```

> `"acceptInsecureCerts"`: true

This instructs the browser to disable certificate validation checks for the session.
